### The Toolkit

- **[scf-api](https://github.com/hackidle/scf-api)** — Static JSON API for the Secure Controls Framework. 1,468 controls, 33 families, 249 framework crosswalks.
- **[nist-cybersecurity-skills](https://github.com/hackidle/nist-cybersecurity-skills)** — Claude Code plugin for NIST SP 800-53, 800-171, CSF 2.0, and more.
- **[homebrew-tap](https://github.com/hackidle/homebrew-tap)** / **[scoop-bucket](https://github.com/hackidle/scoop-bucket)** — Package distribution for hackIDLE tools.
- **[skills](https://github.com/hackidle/skills)** — Security-focused agent skills for mesh analysis, compliance reporting, and remediation.
